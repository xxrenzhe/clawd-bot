## 4. Plugin Marketplace

### Features

| Feature | Description |
|---------|-------------|
| One-click install | Install plugins from marketplace |
| Verified plugins | Security-reviewed plugins |
| Ratings & reviews | Community feedback |
| Auto-updates | Keep plugins current |

### Browse & Install

```bash
# Search marketplace
moltbot plugins search "weather"

# Install from marketplace
moltbot plugins install @community/weather-pro

# View plugin details
moltbot plugins info @community/weather-pro
```

### For Developers

```bash
# Publish your plugin
moltbot plugins publish

# Update existing plugin
moltbot plugins publish --update
```
