## Skills Showcase

### Morning Briefing Skill

```javascript
export default {
  name: 'morning-briefing',
  trigger: 'cron:0 7 * * *',

  async execute(context) {
    const [weather, calendar, news, emails] = await Promise.all([
      context.tools.weather.get(),
      context.tools.calendar.today(),
      context.tools.news.headlines(),
      context.tools.email.unread()
    ]);

    return context.agent.summarize({
      weather,
      calendar,
      news: news.slice(0, 5),
      emailCount: emails.length
    });
  }
};
```
