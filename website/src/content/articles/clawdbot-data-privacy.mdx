## Privacy Configuration

### Enable Privacy Mode

```json
{
  "privacy": {
    "mode": "strict",
    "encryption": true,
    "telemetry": false,
    "logging": "minimal"
  }
}
```

### Data Retention

```json
{
  "privacy": {
    "retention": {
      "conversations": "7d",
      "logs": "1d",
      "cache": "1h"
    }
  }
}
```

## Local-Only Mode

### Use Local LLM

```json
{
  "agent": {
    "provider": "ollama",
    "model": "llama3:70b",
    "local": true
  }
}
```

With Ollama, data never leaves your machine:

```bash
# Install Ollama
curl -fsSL https://ollama.ai/install.sh | sh

# Pull model
ollama pull llama3:70b

# Configure Moltbot
moltbot config set agent.provider ollama
moltbot config set agent.model llama3:70b
```

### Hybrid Mode

Use local LLM for sensitive tasks, cloud for others:

```json
{
  "agent": {
    "providers": {
      "default": {
        "provider": "anthropic",
        "model": "claude-3-5-sonnet"
      },
      "sensitive": {
        "provider": "ollama",
        "model": "llama3:70b"
      }
    },
    "routing": {
      "patterns": {
        "password|secret|private|confidential": "sensitive"
      }
    }
  }
}
```

## Encryption

### At-Rest Encryption

```json
{
  "privacy": {
    "encryption": {
      "enabled": true,
      "algorithm": "AES-256-GCM",
      "keyFile": "~/.moltbot/encryption.key"
    }
  }
}
```

Generate encryption key:
```bash
moltbot security generate-key
```

### Encrypt Specific Data

```json
{
  "privacy": {
    "encryption": {
      "encrypt": ["conversations", "memory", "credentials"],
      "exclude": ["logs", "cache"]
    }
  }
}
```

## Data Minimization

### Reduce Stored Data

```json
{
  "privacy": {
    "minimize": {
      "stripPII": true,
      "truncateMessages": 1000,
      "excludeAttachments": true
    }
  }
}
```

### Automatic PII Redaction

```json
{
  "privacy": {
    "pii": {
      "detect": true,
      "redact": true,
      "patterns": [
        "email",
        "phone",
        "ssn",
        "creditCard"
      ]
    }
  }
}
```

Example:
```
Stored: "Contact me at [EMAIL REDACTED]"
Original: "Contact me at john@example.com"
```

## GDPR Compliance

### Data Subject Rights

```bash
# Export user data
moltbot privacy export --user telegram:123456789 --output user-data.json

# Delete user data
moltbot privacy delete --user telegram:123456789

# Anonymize user data
moltbot privacy anonymize --user telegram:123456789
```

### Consent Management

```json
{
  "privacy": {
    "consent": {
      "required": true,
      "message": "By using this bot, you agree to data processing. Type /privacy for details.",
      "dataTypes": ["conversations", "preferences"]
    }
  }
}
```

## Privacy Checklist

### Initial Setup

| Task | Command |
|------|---------|
| Enable encryption | `moltbot security enable-encryption` |
| Disable telemetry | `moltbot config set privacy.telemetry false` |
| Set retention | `moltbot config set privacy.retention.conversations 7d` |
| Generate keys | `moltbot security generate-key` |

### Regular Maintenance

| Task | Frequency |
|------|-----------|
| Review audit logs | Weekly |
| Purge old data | Automatic |
| Rotate encryption keys | Yearly |
| Review access logs | Monthly |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Encryption key lost | Data cannot be recovered; restore from backup |
| Slow performance with encryption | Use hardware acceleration (AES-NI) |
| PII still visible in logs | Check logging level, enable redaction |
| GDPR export failing | Ensure user ID format is correct |
