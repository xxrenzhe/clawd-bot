## Enable Database Skill

```bash
moltbot skills enable database
```

Configure connection:

```json
{
  "skills": {
    "database": {
      "enabled": true,
      "connections": {
        "production": {
          "type": "postgresql",
          "host": "localhost",
          "port": 5432,
          "database": "myapp",
          "user": "readonly_user",
          "password": "${DB_PASSWORD}"
        }
      }
    }
  }
}
```

## Security Configuration

### Read-Only Access

**Critical:** Always use a read-only database user.

```sql
-- PostgreSQL: Create read-only user
CREATE USER moltbot_readonly WITH PASSWORD 'secure_password';
GRANT CONNECT ON DATABASE myapp TO moltbot_readonly;
GRANT USAGE ON SCHEMA public TO moltbot_readonly;
GRANT SELECT ON ALL TABLES IN SCHEMA public TO moltbot_readonly;
```

```sql
-- MySQL: Create read-only user
CREATE USER 'moltbot_readonly'@'localhost' IDENTIFIED BY 'secure_password';
GRANT SELECT ON myapp.* TO 'moltbot_readonly'@'localhost';
FLUSH PRIVILEGES;
```

### Query Restrictions

```json
{
  "skills": {
    "database": {
      "security": {
        "readOnly": true,
        "maxRows": 1000,
        "timeoutSeconds": 30,
        "blockedTables": ["users_credentials", "api_keys", "sessions"],
        "blockedColumns": ["password_hash", "ssn", "credit_card"]
      }
    }
  }
}
```

### Query Approval

```json
{
  "skills": {
    "database": {
      "security": {
        "requireApproval": true,
        "approvalMessage": "Run this query? (yes/no)"
      }
    }
  }
}
```

## Query Templates

### Pre-defined Queries

```json

    "database": {
      "templates": {
        "daily_signups": "SELECT DATE(created_at), COUNT(*) FROM users WHERE created_at > NOW() - INTERVAL '7 days' GROUP BY 1",
        "revenue_by_product": "SELECT p.name, SUM(o.total) as revenue FROM orders o JOIN products p ON o.product_id = p.id GROUP BY 1 ORDER BY 2 DESC LIMIT 10",
```

## Multiple Databases

### Configure Multiple Connections

```json
{
  "skills": {
    "database": {
      "connections": {
        "production": {
          "type": "postgresql",
          "host": "prod-db.example.com"
        },
        "analytics": {
          "type": "postgresql",
          "host": "analytics-db.example.com"
        },
        "staging": {
          "type": "postgresql",
          "host": "staging-db.example.com"
        }
      },
      "default": "production"
    }
  }
}
```
