## Troubleshooting

| Issue | Solution |
|-------|----------|
| Webhooks not firing | Check webhook secret, verify endpoint |
| Deploy command fails | Verify CI status, check permissions |
| Alerts not sending | Check monitoring provider connection |
| Log search slow | Reduce time range, add filters |

## Security

| Setting | Recommendation |
|---------|----------------|
| Deploy approval | Require confirmation for production |
| Webhook secrets | Rotate every 90 days |
| Shell access | Limit to read-only commands |
| Credentials | Use environment variables only |
