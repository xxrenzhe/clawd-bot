# Moltbot Documentation: AI-Powered Docs Generator

![Moltbot Documentation Generation](/images/articles/clawdbot-documentation.jpg)

## Documentation Types

| Type | Use Case | Output |
|------|----------|--------|
| Code docs | Function/class documentation | JSDoc, docstrings |
| API docs | Endpoint references | OpenAPI, Markdown |
| README | Project overviews | Markdown |
| Guides | Step-by-step tutorials | Markdown, MDX |
| Changelogs | Release notes | CHANGELOG.md |

## Quick Start

```bash
# Document a file
moltbot document src/api/users.ts

# Document entire project
moltbot document ./src --recursive

# Generate README
moltbot readme --from-code
```

## Code Documentation

### Auto-Generate JSDoc

```bash
moltbot document src/utils.js --style jsdoc
```

Before:
```javascript
function calculateTax(amount, rate) {
  return amount * rate;
}
```

After:
```javascript
/**
 * Calculates tax amount based on base amount and tax rate.
 * @param {number} amount - The base amount to calculate tax on
 * @param {number} rate - The tax rate as a decimal (e.g., 0.08 for 8%)
 * @returns {number} The calculated tax amount
 * @example
 * calculateTax(100, 0.08) // Returns 8
 */
function calculateTax(amount, rate) {
  return amount * rate;
}
```

### Python Docstrings

```bash
moltbot document src/api.py --style google
```

Output:
```python
def process_order(order_id: str, items: list[Item]) -> OrderResult:
    """Process an order with the given items.

    Args:
        order_id: Unique identifier for the order.
        items: List of items to include in the order.

    Returns:
        OrderResult containing order status and confirmation.

    Raises:
        OrderError: If order validation fails.
        InventoryError: If items are out of stock.
    """
```

## API Documentation

### Generate OpenAPI Spec

```bash
moltbot api-docs src/routes --format openapi
```

Output:
```yaml
openapi: 3.0.0
info:
  title: User API
  version: 1.0.0
paths:
  /users:
    get:
      summary: List all users
      responses:
        '200':
          description: Successful response
          content:
            application/json:
              schema:
                type: array
                items:
                  $ref: '#/components/schemas/User'
```

### Markdown API Reference

```bash
moltbot api-docs src/routes --format markdown --output docs/api.md
```

## README Generation

### From Code Analysis

```bash
moltbot readme --from-code --include features,install,usage
```

Configuration:
```json
{
  "documentation": {
    "readme": {
      "sections": [
        "badges",
        "description",
        "features",
        "installation",
        "usage",
        "api",
        "contributing",
        "license"
      ],
      "detectFromPackageJson": true
    }
  }
}
```

### Template-Based

```bash
moltbot readme --template enterprise --output README.md
```

## Changelog Generation

### From Git History

```bash
moltbot changelog --from-commits --since v1.0.0
```

Output:
```markdown
## [1.1.0] - 2026-01-28

### Added
- User authentication with OAuth2
- Rate limiting for API endpoints
- Webhook support for external integrations

### Fixed
- Memory leak in long-running sessions
- Incorrect timezone handling in scheduler

### Changed
- Updated Node.js requirement to 22+
- Improved error messages for validation failures
```

```json
    }
  }
}
```

## Configuration

### Documentation Settings

```json
{
  "documentation": {
    "style": "jsdoc",
    "language": "en",
    "output": "./docs",
    "templates": {
      "function": "custom/function.md",
      "class": "custom/class.md"
    },
    "ignore": [
      "node_modules",
      "dist",
      "*.test.js"
    ]
  }
}
```

## Batch Documentation

### Document Project

```bash
# Full project documentation
moltbot document . --recursive --output docs/

# Specific directories
moltbot document src/api src/utils --output docs/reference/

# Watch mode
moltbot document src/ --watch
```

### CI Integration

```yaml
# .github/workflows/docs.yml
name: Documentation
on:
  push:
    branches: [main]

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Generate docs
        run: moltbot document src/ --output docs/
      - name: Deploy to Pages
        uses: peaceiris/actions-gh-pages@v3
        with:
          publish_dir: ./docs
```

## Troubleshooting

| Issue | Cause | Solution |
|-------|-------|----------|
| Missing types | No TypeScript | Use --infer-types |
| Wrong format | Bad config | Check style setting |
| Skipped files | Ignore patterns | Review ignore list |
| Incomplete docs | Private members | Use --include-private |

### Debug Mode

```bash
moltbot document src/ --verbose
```
