## Gmail Setup

### Step 1: Enable Gmail API

**Option A: App Password (simpler)**

1. Go to [myaccount.google.com/security](https://myaccount.google.com/security)
2. Enable 2-Step Verification
3. Go to "App passwords"
4. Generate new app password for "Mail"
5. Copy the 16-character password

**Option B: OAuth (recommended for security)**

1. Go to [console.cloud.google.com](https://console.cloud.google.com)
2. Create new project
3. Enable Gmail API
4. Create OAuth credentials
5. Download `credentials.json`

### Step 2: Install Email Skill

```bash
moltbot skill add email
```

### Step 3: Configure

```bash
moltbot configure --section skills.email
```

<HostingCTA context="inline" />

For app password:
```json
{
  "skills": {
    "email": {
      "provider": "gmail",
      "email": "you@gmail.com",
      "appPassword": "xxxx xxxx xxxx xxxx"
    }
  }
}
```

For OAuth:
```json
{
  "skills": {
    "email": {
      "provider": "gmail-oauth",
      "credentialsPath": "~/.moltbot/gmail-credentials.json"
    }
  }
}
```

Run OAuth flow:
```bash
moltbot auth gmail
```

## Using Email Commands

### Summarize Inbox

```
You: Summarize my unread emails
Moltbot: You have 12 unread emails:

         **Urgent (2):**
         - Client X: Contract deadline tomorrow
         - Boss: Need project update by EOD

         **Newsletters (5):**
         - Tech Daily, Product Hunt, etc.

         **Normal (5):**
         - Team meeting notes
         - Invoice from vendor
         - 3 GitHub notifications
```

### Draft Reply

```
You: Draft a reply to the contract email saying I'll review by tomorrow
Moltbot: **Draft ready:**

         Subject: Re: Contract Review

         Hi Sarah,

         Thank you for sending the contract. I'll review it
         thoroughly and send my feedback by tomorrow EOD.

         Best regards,
         [Your name]

         ---
         Send now? [Yes/No]
```

## Security Considerations

| Setting | Recommendation |
|---------|----------------|
| Authentication | Use OAuth over app passwords |
| Email account | Consider dedicated account for AI |
| Permissions | Read-only initially, add send later |
| Sensitive emails | Configure exclusion patterns |

### Exclude Sensitive Emails

```json
{
  "skills": {
    "email": {
      "exclude": {
        "senders": ["hr@company.com", "legal@company.com"],
        "subjects": ["confidential", "personal"]
      }
    }
  }
}
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| "Authentication failed" | Regenerate app password or re-run OAuth |
| "Rate limited" | Reduce request frequency |
| "Permission denied" | Check API scopes are granted |
| "Email not found" | Check search query, try broader terms |

**Test connection:**
```bash
moltbot skill test email
```

```json

  "skills": {
    "email": {
      "rules": [
        {
          "match": "from:boss@company.com",
          "action": "notify_urgent"
        },
        {
          "match": "subject:invoice",
          "action": "label:Finance"
        }
      ]
    }
  }
}
```
