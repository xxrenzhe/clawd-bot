# Moltbot Embeddings: Semantic Search & RAG

![Moltbot Embeddings](/images/articles/clawdbot-embeddings.jpg)

## Embeddings Overview

| Provider | Model | Dimensions | Best For |
|----------|-------|------------|----------|
| OpenAI | text-embedding-3-large | 3072 | General purpose |
| OpenAI | text-embedding-3-small | 1536 | Cost-effective |
| Voyage | voyage-2 | 1024 | Code, technical |
| Local | nomic-embed | 768 | Privacy, offline |

## Enable Embeddings

```bash
moltbot skills enable embeddings
```

Configure provider:

```json
{
  "embeddings": {
    "provider": "openai",
    "model": "text-embedding-3-small",
    "dimensions": 1536
  }
}
```

## Vector Store Setup

### Built-in SQLite Store

```json
{
  "embeddings": {
    "store": {
      "type": "sqlite",
      "path": "~/.moltbot/vectors.db"
    }
  }
}
```

### Qdrant (Production)

```json
{
  "embeddings": {
    "store": {
      "type": "qdrant",
      "url": "http://localhost:6333",
      "collection": "moltbot_docs"
    }
  }
}
```

### Pinecone (Cloud)

```json
{
  "embeddings": {
    "store": {
      "type": "pinecone",
      "apiKey": "${PINECONE_API_KEY}",
      "environment": "us-east-1",
      "index": "moltbot"
    }
  }
}
```

## Index Documents

### Add Documents

```bash
# Index a directory
moltbot embeddings index ~/docs

# Index specific files
moltbot embeddings index ~/docs/api.md ~/docs/guide.md

# Index with metadata
moltbot embeddings index ~/docs --metadata '{"source": "documentation"}'
```

## RAG Implementation

### Enable RAG

```json
{
  "agent": {
    "rag": {
      "enabled": true,
      "sources": ["documentation", "knowledge-base"],
      "topK": 5,
      "minSimilarity": 0.7
    }
  }
}
```

### RAG Workflow

```
User: "How do I deploy to Docker?"

1. Query embedded ‚Üí Vector search
2. Retrieve top 5 relevant chunks
3. Inject into context
4. LLM generates answer with sources
```

Example:

## Knowledge Base

### Create Knowledge Base

```bash
moltbot kb create company-docs
```

### Add Content

```bash
# Add files
moltbot kb add company-docs ~/docs/policies/*.pdf

# Add URL
moltbot kb add company-docs https://docs.example.com/api

# Add text directly
moltbot kb add company-docs --text "Our return policy is 30 days..."
```

### Query Knowledge Base

```
You: What's our vacation policy?

Moltbot: üîç Searching company-docs...

According to the Employee Handbook (section 4.2):

"Employees receive 15 days of paid vacation annually,
accruing at 1.25 days per month. Unused vacation
carries over up to 5 days into the next year."

Source: employee-handbook.pdf, page 12
```

## Hybrid Search

Combine vector search with keyword matching:

```json
{
  "embeddings": {
    "search": {
      "type": "hybrid",
      "vectorWeight": 0.7,
      "keywordWeight": 0.3
    }
  }
}
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Low similarity scores | Increase chunk overlap |
| Slow indexing | Use batch processing |
| Out of memory | Reduce chunk size |
| Missing context | Increase topK results |

### Debug Search

```bash
# Verbose search
moltbot embeddings search "authentication" --verbose

# Check index
moltbot embeddings verify

# Rebuild index
moltbot embeddings reindex
```

## Best Practices

| Practice | Benefit |
|----------|---------|
| Chunk by semantic units | Better retrieval |
| Include metadata | Filter results |
| Use hybrid search | Handle edge cases |
| Cache embeddings | Reduce costs |
| Regular reindexing | Fresh content |
