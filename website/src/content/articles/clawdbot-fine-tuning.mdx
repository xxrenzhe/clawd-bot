# Moltbot Fine-Tuning: Customize AI Behavior

![Moltbot Fine-Tuning](/images/articles/clawdbot-fine-tuning.jpg)

## Customization Options

| Method | Complexity | Effect |
|--------|------------|--------|
| System Prompt | Low | Personality, rules |
| Personas | Medium | Role-specific behavior |
| Response Templates | Medium | Output format |
| Tool Configuration | High | Capability limits |

## System Prompt

### Basic Configuration

```json
{
  "agent": {
    "systemPrompt": "You are a helpful AI assistant. Be concise, accurate, and friendly. Always cite sources when providing factual information."
  }
}
```

### Detailed System Prompt

```json
{
  "agent": {
    "systemPrompt": {
      "role": "You are Aria, a personal productivity assistant.",
      "personality": "Friendly, efficient, and proactive.",
      "rules": [
        "Keep responses under 200 words unless asked for detail",
        "Always suggest next actions when completing tasks",
        "Use bullet points for lists",
        "Never share personal opinions on controversial topics"
      ],
      "knowledge": [
        "Expert in time management",
        "Familiar with GTD methodology",
        "Knowledgeable about productivity tools"
      ]
    }
  }
}
```

## Personas

### Define Personas

```json
{
  "agent": {
    "personas": {
      "developer": {
        "name": "DevBot",
        "systemPrompt": "You are a senior software engineer. Provide technical, code-focused responses. Include code examples when relevant.",
        "model": "claude-3-5-sonnet",
        "temperature": 0.3
      },
      "writer": {
        "name": "Wordsmith",
        "systemPrompt": "You are a creative writing assistant. Help with drafts, editing, and brainstorming. Be creative and encouraging.",
        "model": "claude-3-5-sonnet",
        "temperature": 0.8
      },
      "analyst": {
        "name": "DataPro",
        "systemPrompt": "You are a data analyst. Focus on numbers, trends, and insights. Be precise and methodical.",
        "model": "claude-3-5-sonnet",
        "temperature": 0.2
      }
    },
    "defaultPersona": "developer"
  }
}
```

## Response Styles

### Verbosity

```json
{
  "agent": {
    "style": {
      "verbosity": "concise",
      "maxResponseLength": 500,
      "useMarkdown": true,
      "useBulletPoints": true
    }
  }
}
```

Verbosity options:
- `minimal`: One-liner responses
- `concise`: Brief but complete
- `detailed`: Thorough explanations
- `comprehensive`: Deep dives with examples

### Formatting

```json
{
  "agent": {
    "formatting": {
      "codeBlocks": true,
      "tables": true,
      "emoji": "minimal",
      "headers": true
    }
  }
}
```

### Tone

```json
{
  "agent": {
    "tone": {
      "formality": "casual",
      "humor": "light",
      "encouragement": true
    }
  }
}
```

## Temperature & Creativity

```json
{
  "agent": {
    "generation": {
      "temperature": 0.7,
      "topP": 0.9,
      "topK": 40
    }
  }
}
```

| Temperature | Use Case |
|-------------|----------|
| 0.0-0.3 | Code, facts, analysis |
| 0.4-0.6 | Balanced responses |
| 0.7-0.9 | Creative writing |
| 1.0+ | Brainstorming, poetry |

## Task-Specific Tuning

### Code Assistant

```json
{
  "agent": {
    "modes": {
      "coding": {
        "temperature": 0.2,
        "systemPrompt": "You are a code-focused assistant. Always provide working code examples. Explain briefly, show code first.",
        "tools": ["code_execute", "file_read", "file_write"],
        "formatting": {
          "codeBlocks": true,
          "preferredLanguages": ["python", "javascript", "typescript"]
        }
      }
    }
  }
}
```

### Research Assistant

  "agent": {

        "tools": ["web_search", "web_fetch"],
        "formatting": {
          "citations": true,

## Response Templates

### Define Templates

```json
{
  "agent": {
    "templates": {
      "bug_report": {
        "format": "## Bug Summary\n{summary}\n\n## Steps to Reproduce\n{steps}\n\n## Expected vs Actual\n{comparison}\n\n## Environment\n{environment}"
      },
      "meeting_notes": {
        "format": "# {title}\nDate: {date}\n\n## Attendees\n{attendees}\n\n## Key Points\n{points}\n\n## Action Items\n{actions}"
      }
    }
  }
}
```

### Use Templates

```
You: /template bug_report

Moltbot: I'll format your bug report. Please provide:
1. Summary of the bug
2. Steps to reproduce
3. What you expected vs what happened
4. Your environment (OS, browser, version)
```

## Knowledge Injection

### Custom Knowledge

```json
{
  "agent": {
    "knowledge": {
      "company": {
        "name": "Acme Corp",
        "products": ["Widget Pro", "Widget Lite"],
        "policies": {
          "refund": "30-day money-back guarantee",
          "support": "24/7 email, 9-5 phone"
        }
      }
    }
  }
}
```

### Inject at Runtime

```json
{
  "agent": {
    "contextInjection": {
      "enabled": true,
      "sources": [
        { "type": "file", "path": "~/company/policies.md" },
        { "type": "url", "url": "https://internal.acme.com/faq" }
      ]
    }
  }
}
```

## Troubleshooting

| Issue | Cause | Solution |
|-------|-------|----------|
| Ignores instructions | Prompt too long | Prioritize key rules |
| Too verbose | Wrong verbosity setting | Set maxResponseLength |
| Wrong persona | Default not set | Check defaultPersona |
| Inconsistent tone | Temperature too high | Lower temperature |

### Debug Prompts

```bash
# View effective system prompt
moltbot config show --section agent.systemPrompt

# Test prompt
moltbot chat "Hello" --debug-prompt
```
