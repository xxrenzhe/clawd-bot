# Moltbot First Run: Your First Message in 5 Minutes

![Moltbot First Run](/images/articles/clawdbot-first-run.jpg)

## Quick Start Checklist

| Step | Command | Time |
|------|---------|------|
| 1. Verify install | `moltbot --version` | 10 sec |
| 2. Run onboarding | `moltbot onboard` | 2 min |
| 3. Start gateway | `moltbot gateway` | 30 sec |
| 4. Send first message | Via Telegram/Discord | 1 min |

## Prerequisites

| Requirement | Check |
|-------------|-------|
| Moltbot installed | `moltbot --version` shows 2.x+ |
| API key ready | Anthropic, OpenAI, or Gemini |
| Bot token (optional) | For Telegram/Discord |

## Step 1: Verify Installation

```bash
moltbot --version
```

Expected output:
```
moltbot v2.4.1
```

If not installed, run:
```bash
curl -fsSL https://molt.bot/install.sh | bash
```

## First Run Configuration

Your configuration is saved at `~/.moltbot/moltbot.json`:

```json
{
  "gateway": {
    "port": 18789,
    "bind": "loopback"
  },
  "agent": {
    "provider": "anthropic",
    "model": "claude-3-5-sonnet",
    "systemPrompt": "You are a helpful personal assistant."
  },
  "channels": {
    "telegram": {
      "enabled": true,
      "allowedUsers": ["your_username"]
    }
  }
}
```

## Common First-Run Issues

| Problem | Solution |
|---------|----------|
| "Command not found" | Run the installer again, restart terminal |
| "API key invalid" | Verify key at provider's dashboard |
| "Port in use" | `moltbot gateway --port 18790` |
| "Bot not responding" | Check bot token, verify gateway is running |
| "Permission denied" | Run `chmod +x ~/.moltbot/bin/moltbot` |
