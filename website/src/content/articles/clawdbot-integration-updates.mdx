## API Connections

### Custom API Integration

```json
{
  "integrations": {
    "custom": {
      "my-api": {
        "baseUrl": "https://api.example.com",
        "auth": {
          "type": "bearer",
          "token": "${MY_API_TOKEN}"
        },
        "endpoints": {
          "getUsers": { "method": "GET", "path": "/users" },
          "createOrder": { "method": "POST", "path": "/orders" }
        }
      }
    }
  }
}
```

## Getting Started

```bash
# Enable an integration
moltbot integrations enable notion

# Configure credentials
moltbot configure --section integrations.notion

# Test connection
moltbot integrations test notion
```
