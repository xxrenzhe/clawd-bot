# Moltbot Logging: Monitor & Debug Setup

![Moltbot Logging Practices](/images/articles/clawdbot-logging-practices.jpg)

## Log Levels

| Level | Use Case | Example |
|-------|----------|---------|
| `error` | Failures requiring attention | API errors, crashes |
| `warn` | Potential issues | Rate limit approaching |
| `info` | Normal operations | Requests, responses |
| `debug` | Troubleshooting | Request details |
| `trace` | Deep debugging | Token-level data |

## Basic Configuration

```json
{
  "logging": {
    "level": "info",
    "format": "pretty",
    "timestamp": true
  }
```

## File Logging

### Single File

```json
{
  "logging": {
    "file": {
      "enabled": true,
      "path": "~/.moltbot/logs/moltbot.log",
      "level": "info"
    }
  }
}
```

### Separate Log Files

```json
{
  "logging": {
    "files": {
      "all": {
        "path": "~/.moltbot/logs/all.log",
        "level": "info"
      },
      "errors": {
        "path": "~/.moltbot/logs/errors.log",
        "level": "error"
      },
      "debug": {
        "path": "~/.moltbot/logs/debug.log",
        "level": "debug"
      }
    }
  }
}
```

## Log Rotation

### Size-Based Rotation

```json
{
  "logging": {
    "rotation": {
      "enabled": true,
      "maxSize": "10MB",
      "maxFiles": 5,
      "compress": true
    }
  }
}
```

### Time-Based Rotation

```json
{
  "logging": {
    "rotation": {
      "enabled": true,
      "frequency": "daily",
      "maxAge": "7d",
      "compress": true
    }
  }
}
```

## Structured Logging

### JSON Format

```json
{
  "logging": {
    "format": "json",
    "fields": {
      "service": "moltbot",
      "environment": "production"
    }
  }
}
```

Output:
```json
{"timestamp":"2026-01-28T14:30:45Z","level":"info","message":"Request processed","service":"moltbot","requestId":"abc123","channel":"telegram","userId":"user_456","duration":1234}
```

### Custom Fields

```json
{
  "logging": {
    "format": "json",
    "includeFields": [
      "requestId",
      "channel",
      "userId",
      "model",
      "tokens",
      "duration"
    ]
  }
}
```

## View Logs

### CLI Commands

```bash
# View recent logs
moltbot logs --tail 100

# Follow logs in real-time
moltbot logs -f

# Filter by level
moltbot logs --level error

# Filter by component
moltbot logs --filter "channel=telegram"

# Time range
moltbot logs --since "1h" --until "now"
```

### Search Logs

```bash
# Search for errors
moltbot logs --grep "error" --level error

# Search by request ID
moltbot logs --request-id abc123

# Search by user
moltbot logs --user user_456
```

## Debug Mode

### Enable Debug

```bash
# Environment variable
MOLTBOT_DEBUG=true moltbot gateway

# CLI flag
moltbot gateway --debug

# Specific component
MOLTBOT_DEBUG=agent,tools moltbot gateway
```

## Sensitive Data

### Redaction

```json
{
  "logging": {
    "redact": {
      "enabled": true,
      "patterns": [
        "apiKey",
        "token",
        "password",
        "secret"
      ],
      "replacement": "[REDACTED]"
    }
  }
}
```

### PII Protection

```json
{
  "logging": {
    "privacy": {
      "hashUserIds": true,
      "excludeMessageContent": true,
      "maskIpAddresses": true
    }
  }
}
```

## Troubleshooting

| Issue | Cause | Solution |
|-------|-------|----------|
| Logs too verbose | Debug level enabled | Set level to "info" |
| Missing logs | Wrong path | Check file permissions |
| Disk full | No rotation | Enable log rotation |
| Slow performance | Sync writes | Use async logging |
| Secrets in logs | No redaction | Enable redact patterns |
