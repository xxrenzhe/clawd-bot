# Moltbot on Mac Mini: 24/7 AI Assistant Setup

![Moltbot running 24/7 on Mac Mini](/images/articles/clawdbot-mac-mini-setup.jpg)

## Why Mac Mini for Moltbot?

Mac Mini is ideal for running Moltbot (formerly Clawdbot) 24/7:

| Feature | Mac Mini M4 | VPS | Raspberry Pi |
|---------|-------------|-----|--------------|
| Power consumption | 5-7W idle | N/A | 3-5W |
| Performance | Excellent | Good | Limited |
| iMessage support | Yes | No | No |
| Monthly cost | ~$2 electricity | $5-20 | ~$1 electricity |
| Setup effort | Low | Medium | Medium |

Mac Mini + Moltbot = Your personal AI that's always available and supports Apple ecosystem features.

## Step 1: Install Moltbot

```bash
# Install Homebrew if needed
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install Moltbot
brew install moltbot

# Run setup
moltbot onboard
```

Get your API key from [console.anthropic.com](https://console.anthropic.com).

## Step 2: Configure Power Settings

Prevent your Mac from sleeping:

**System Settings → Energy:**
- Turn off "Put hard disks to sleep when possible"
- Set "Turn display off after" to 5 minutes (display can sleep)
- Enable "Wake for network access"
- Enable "Start up automatically after power failure"

**Via Terminal:**
```bash
# Prevent sleep (indefinitely)
sudo pmset -a sleep 0
sudo pmset -a disksleep 0

# Wake on network access
sudo pmset -a womp 1

# Restart after power failure
sudo pmset -a autorestart 1

# Verify settings
pmset -g
```

## Step 3: Create LaunchAgent (Auto-Start)

Create the plist file:

```bash
mkdir -p ~/Library/LaunchAgents
nano ~/Library/LaunchAgents/com.moltbot.gateway.plist
```

Paste this content:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.moltbot.gateway</string>
    <key>ProgramArguments</key>
    <array>
        <string>/opt/homebrew/bin/moltbot</string>
        <string>gateway</string>
    </array>
    <key>RunAtLoad</key>
    <true/>
    <key>KeepAlive</key>
    <true/>
    <key>StandardOutPath</key>
    <string>/tmp/moltbot.log</string>
    <key>StandardErrorPath</key>
    <string>/tmp/moltbot.error.log</string>
    <key>EnvironmentVariables</key>
    <dict>
        <key>ANTHROPIC_API_KEY</key>
        <string>sk-ant-your-key-here</string>
    </dict>
</dict>
</plist>
```

Load the agent:
```bash
launchctl load ~/Library/LaunchAgents/com.moltbot.gateway.plist
```

## Step 4: Enable Automatic Login

For 24/7 operation, Mac needs to log in automatically after restarts:

**System Settings → Users & Groups → Login Options:**
- Set "Automatic login" to your user
- Consider using a dedicated Moltbot user account for security

## Step 5: Set Up Remote Access

### Option A: Tailscale (Recommended)

```bash
brew install tailscale
open /Applications/Tailscale.app
```

1. Sign in with your Tailscale account
2. Access your Mac Mini from anywhere via Tailscale IP
3. No port forwarding needed, secure by default

### Option B: SSH

```bash
# Enable Remote Login
sudo systemsetup -setremotelogin on
```

Access from another device:
```bash
ssh username@mac-mini.local
```

### Option C: Screen Sharing

**System Settings → General → Sharing:**
- Enable "Screen Sharing"
- Use with Tailscale for secure remote access

## Step 6: Connect iMessage (Mac-Exclusive)

One major advantage of Mac Mini: iMessage integration.

```bash
moltbot configure --section channels.imessage
```

Now you can message your AI assistant via iMessage from any Apple device.

## Step 7: Verify 24/7 Operation

Check status:
```bash
# Check if gateway is running
moltbot gateway status

# View logs
tail -f /tmp/moltbot.log

# Check LaunchAgent status
launchctl list | grep moltbot
```

Test restart recovery:
1. Restart your Mac Mini
2. After login, check `moltbot gateway status`
3. Should show "running"

## Monitoring and Maintenance

### Daily health check (optional cron)

```bash
crontab -e
```

Add:
```
0 9 * * * /opt/homebrew/bin/moltbot health >> /tmp/moltbot-health.log 2>&1
```

### Update Moltbot

```bash
brew upgrade moltbot
launchctl unload ~/Library/LaunchAgents/com.moltbot.gateway.plist
launchctl load ~/Library/LaunchAgents/com.moltbot.gateway.plist
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Gateway doesn't start after reboot | Check `launchctl list | grep moltbot`, verify plist syntax |
| Mac keeps sleeping | Run `pmset -g` to verify power settings |
| Can't connect remotely | Enable "Wake for network access" in Energy settings |
| High CPU usage | Check `/tmp/moltbot.log` for errors |
| LaunchAgent not loading | `launchctl unload` then `load` again |

**View LaunchAgent errors:**
```bash
cat /tmp/moltbot.error.log
```

**Restart the gateway:**
```bash
launchctl unload ~/Library/LaunchAgents/com.moltbot.gateway.plist
launchctl load ~/Library/LaunchAgents/com.moltbot.gateway.plist
```
