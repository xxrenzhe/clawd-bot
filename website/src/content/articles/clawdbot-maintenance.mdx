## Updates

### Check for Updates

```bash
moltbot update --check
```

Output:
```
üì¶ Update Available

Current: 2.3.4
Latest: 2.4.0

Changes:
- New: Improved streaming performance
- Fix: Memory leak in long sessions
- Fix: Discord reconnection issue

```

### Apply Updates

```bash
# Update to latest
moltbot update

# Update to specific version
moltbot update --version 2.4.0

# Update with backup
moltbot update --backup
```

## Backups

### Manual Backup

```bash
# Full backup
moltbot backup create

# Backup to specific location
moltbot backup create --output ~/backups/moltbot-$(date +%Y%m%d).tar.gz

# Include conversation history
moltbot backup create --include-history
```

### Backup Contents


### Automated Backups

```json
{
  "backup": {
    "enabled": true,
    "schedule": "0 2 * * *",
    "retention": 7,
    "destination": {
      "type": "s3",
      "bucket": "moltbot-backups",
      "region": "us-east-1"
    },
    "encrypt": true
  }
}
```

### Restore

```bash
# List available backups
moltbot backup list

# Restore from backup
moltbot backup restore ./backup-2026-01-28.tar.gz

# Restore specific components
moltbot backup restore ./backup.tar.gz --only config,plugins
```

## Log Management

### View Logs

```bash
# Recent logs
moltbot logs --tail 100

# Filter by level
moltbot logs --level error --since "24h"

# Follow logs
moltbot logs -f
```

### Log Rotation

```json
{
  "logging": {
    "rotation": {
      "enabled": true,
      "maxSize": "10MB",
      "maxFiles": 5,
      "compress": true
    }
  }
}
```

### Clean Old Logs

```bash
# Remove logs older than 7 days
moltbot logs clean --older-than 7d

# Remove all debug logs
moltbot logs clean --level debug
```

## Database Maintenance

### Optimize Database

```bash
# Vacuum SQLite
moltbot db optimize

# Rebuild indexes
moltbot db reindex

# Check integrity
moltbot db check
```

### Prune Old Data

```json
{
  "maintenance": {
    "prune": {
      "enabled": true,
      "schedule": "0 4 * * SUN",
      "rules": {
        "conversations": { "olderThan": "90d" },
        "embeddings": { "unusedFor": "30d" },
        "logs": { "olderThan": "7d" }

  }
```

## Maintenance Mode

### Enable Maintenance

```bash
# Enter maintenance mode
moltbot maintenance on --message "Updating... Back in 5 minutes"
```

Users see:
```

```

### Disable Maintenance

```bash
moltbot maintenance off
```

## Troubleshooting

| Issue | Cause | Solution |
|-------|-------|----------|
| High memory | Memory leak | Restart, update |
| Slow responses | Cache miss | Warm cache |
| Update fails | Dependencies | Check Node.js version |
| Backup fails | Disk space | Free space |

### Quick Fixes

```bash
# Restart gateway
moltbot gateway restart

# Reset to defaults
moltbot config reset --keep-keys

# Repair installation
moltbot repair

# Full diagnostic
moltbot doctor --fix
```
