# Moltbot Meeting Notes: Transcription, Summaries, Actions

![Moltbot Meeting Notes](/images/articles/clawdbot-meeting-notes.jpg)

## Meeting Capabilities

| Feature | Description | Output |
|---------|-------------|--------|
| Transcription | Audio to text | Full transcript |
| Summarization | Key points extraction | 1-page summary |
| Action items | Task identification | Assignee + deadline |
| Follow-ups | Send to attendees | Email/Slack |
| Integration | Sync to Notion/docs | Searchable notes |

<HostingCTA context="setup" />

## Prerequisites

| Requirement | Notes |
|-------------|-------|
| Moltbot installed | `moltbot health` |
| Calendar skill | For meeting context |
| Transcription API | Whisper or AssemblyAI |

## Quick Setup

```bash
moltbot skill add calendar
moltbot skill add transcription
moltbot skill add notion  # optional
```

## Calendar Integration

### Pre-Meeting Prep

```json
{
  "triggers": {
    "meeting_prep": {
      "event": "calendar.meeting.upcoming",
      "conditions": {
        "minutes_before": 5
      },
      "action": {
        "prompt": "Prepare meeting context for {{title}} with {{attendees}}"
      }
    }
  }
}
```

5 minutes before meeting:
```
üìÖ Meeting starting in 5 minutes:
"Weekly 1:1 with Sarah"

**Previous meeting notes (Jan 21):**
- Discussed Q1 goals
- Sarah mentioned workload concerns
- Action: You committed to reviewing her project timeline

**Open items from last time:**
- [ ] Review project timeline (assigned to you)
- [x] Submit budget request (completed Jan 23)

**Suggested topics:**
- Follow up on workload
- Q1 goal progress check
- Any blockers?

Good luck! üéØ
```

### Post-Meeting Automation

```json
{
  "triggers": {
    "meeting_ended": {
      "event": "calendar.meeting.ended",
      "conditions": {
        "hasRecording": true
      },
      "action": {
        "tasks": ["transcribe", "summarize", "extract_actions", "send_followup"]
      }
    }
  }
}
```

```json
{

        "database": "Meeting Notes",
        "template": "meeting-template",
```

## Configuration

```json
{
  "skills": {
    "meetings": {
      "transcription": {
        "provider": "whisper",
        "language": "auto",
        "speakerDiarization": true
      },
      "summary": {
        "style": "concise",
        "includeTimestamps": true,
        "extractActions": true
      },
      "followup": {
        "autoSend": false,
        "channel": "email",
        "template": "meeting-followup"
      },
      "storage": {
        "provider": "notion",
        "database": "Meeting Notes"
      }
    }
  }
}
```

## Best Practices

| Practice | Recommendation |
|----------|----------------|
| Recording quality | Use dedicated mic, quiet room |
| Speaker identification | Ask participants to state name at start |
| Action items | Be explicit: "I'll do X by Y" |
| Follow-up timing | Send within 1 hour of meeting |
| Review | Scan AI summary for accuracy |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Poor transcription | Improve audio quality, check mic |
| Wrong speakers | Enable speaker diarization |
| Missing actions | Use clearer commitment language |
| Summary too long | Set `style: "brief"` in config |

## Privacy Notes

| Setting | Recommendation |
|---------|----------------|
| Recording consent | Always inform attendees |
| Storage | Use encrypted providers |
| Retention | Set auto-delete policy |
| Access | Limit to meeting attendees |
