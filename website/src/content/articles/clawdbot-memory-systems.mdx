# Moltbot Memory: Persistent AI Context

![Moltbot Memory Systems](/images/articles/clawdbot-memory-systems.jpg)

## Memory Types

| Type | Duration | Use Case | Storage |
|------|----------|----------|---------|
| Working | Session | Current conversation | In-memory |
| Short-term | Hours/Days | Recent context | Cache |
| Long-term | Persistent | Facts, preferences | Database |
| Episodic | Permanent | Past conversations | Vector DB |

## Enable Memory

```json
{
  "agent": {
    "memory": {
      "enabled": true,
      "types": ["working", "shortTerm", "longTerm"],
      "storage": "sqlite"
    }
  }
}
```

## Working Memory

### Configuration

```json
{
  "agent": {
    "memory": {
      "working": {
        "maxMessages": 50,
        "maxTokens": 8000,
        "summarizeThreshold": 30
      }
    }
  }
}
```

### Auto-Summarization

When conversation exceeds threshold:

```
[Memory: Summarizing older messages]

Summary of previous conversation:
- User asked about Python async programming
- Discussed asyncio basics and event loops
- User shared code for a web scraper
- Identified bug in error handling

[Continuing with recent context]
```

## Short-Term Memory

### Configuration

```json
{
  "agent": {
    "memory": {
      "shortTerm": {
        "duration": "24h",
        "maxEntries": 100,
        "storage": "redis"
      }
    }
  }
}
```

### Redis Storage

```json
{
  "agent": {
    "memory": {
      "shortTerm": {
        "redis": {
          "url": "redis://localhost:6379",
          "prefix": "moltbot:memory:",
          "ttl": 86400
        }
      }
    }
  }
}
```

### SQLite Storage

```json
{

      "longTerm": {
        "storage": "sqlite",
        "path": "~/.moltbot/memory.db",
        "categories": ["facts", "preferences", "projects"]

    }
  }

### PostgreSQL Storage

```json
{
  "agent": {
    "memory": {

        "storage": "postgres",

  }

## Episodic Memory

### Vector-Based Recall

```json
{
  "agent": {
    "memory": {
      "episodic": {
        "enabled": true,
        "vectorStore": "chroma",
        "embeddings": "openai",
        "similarityThreshold": 0.75
      }
    }
  }
}
```

### Recall Past Conversations

```
You: What did we discuss about Docker last week?

Moltbot: [Searching episodic memory...]

Found 3 relevant conversations:

1. **Jan 22** - Docker multi-stage builds
   - Discussed optimizing image size
   - You created a Python Dockerfile

2. **Jan 24** - Docker Compose networking
   - Set up a multi-container app
   - Resolved port conflict issue

3. **Jan 25** - Docker volumes
   - Explained bind mounts vs volumes
   - Fixed data persistence issue
```

### Manage Memory

```bash
# Add memory
moltbot memory add --user user_123 --fact "Uses macOS"

# Remove memory
moltbot memory remove --user user_123 --id mem_abc123

# Clear short-term
moltbot memory clear --user user_123 --type short-term

# Export memory
moltbot memory export --user user_123 --format json > memory.json
```

## Troubleshooting

| Issue | Cause | Solution |
|-------|-------|----------|
| Forgetting context | Low token limit | Increase maxTokens |
| Wrong recall | Low similarity threshold | Adjust threshold |
| Slow responses | Large memory | Enable summarization |
| Privacy concerns | Default retention | Set shorter retention |

### Debug Memory

```bash
# Check memory status
moltbot memory status

# View memory queries
moltbot memory debug --user user_123

# Rebuild index
moltbot memory reindex
```
