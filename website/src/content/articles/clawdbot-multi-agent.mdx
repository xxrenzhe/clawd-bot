## Orchestrator Agent

### Configuration

```json
{
  "agent": {
    "orchestrator": {
      "enabled": true,
      "model": "claude-3-5-sonnet",
      "systemPrompt": "You are the orchestrator. Break complex tasks into subtasks, delegate to specialist agents, and synthesize results.",
      "canDelegate": ["researcher", "coder", "writer", "reviewer"],
      "maxDelegations": 10
    }
  }
}
```

### Delegation Flow

```
User: "Create a blog post about GraphQL vs REST APIs"

Orchestrator: I'll coordinate this task.

1. [Delegating to researcher] Research GraphQL vs REST comparison
2. [Delegating to coder] Create code examples for both
3. [Delegating to writer] Draft the blog post
4. [Delegating to reviewer] Review and polish

[researcher]: Found 5 key differences, performance benchmarks...
[coder]: Created 3 code examples for each approach...
[writer]: Drafted 1,500-word article with examples...
[reviewer]: Suggested 4 improvements, fixed 2 errors...

Orchestrator: Here's your completed blog post:
[Final synthesized content]
```

## Sequential Workflow

### Define Workflow

```json
{
  "workflows": {
    "content_pipeline": {
      "type": "sequential",
      "steps": [
        {
          "agent": "researcher",
          "task": "Research the topic",
          "output": "research_notes"
        },
        {
          "agent": "writer",
          "task": "Draft content using research",
          "input": "research_notes",
          "output": "draft"
        },
        {
          "agent": "reviewer",
          "task": "Review and suggest edits",
          "input": "draft",
          "output": "feedback"
        },
        {
          "agent": "writer",
          "task": "Apply feedback and finalize",
          "input": ["draft", "feedback"],
          "output": "final"
        }
      ]
    }
  }
}
```

### Run Workflow

```bash
moltbot workflow run content_pipeline --input "AI in healthcare"
```

## Parallel Execution

### Parallel Tasks

```json
{
  "workflows": {
    "multi_research": {
      "type": "parallel",
      "tasks": [
        {
          "agent": "researcher",
          "task": "Research competitor A",
          "output": "competitor_a"
        },
        {
          "agent": "researcher",
          "task": "Research competitor B",
          "output": "competitor_b"
        },
        {
          "agent": "researcher",
          "task": "Research market trends",
          "output": "trends"
        }
      ],
      "merge": {
        "agent": "writer",
        "task": "Synthesize research into report",
        "input": ["competitor_a", "competitor_b", "trends"]
```

### Team Structure

```json
  "agent": {
    "teams": {
      "engineering": {
        "lead": "senior_dev",
```

### Delegation Rules

```json
{
  "agent": {
    "delegation": {
      "rules": [
        {
          "pattern": "code|programming|debug",
          "delegate": "engineering"
        },

          "delegate": "content"

  }
}
```

## Troubleshooting

| Issue | Cause | Solution |
|-------|-------|----------|
| Slow coordination | Too many agents | Reduce concurrent limit |
| Conflicting outputs | No clear ownership | Define responsibilities |
| Infinite loops | Circular delegation | Set max delegations |
| High costs | Parallel overhead | Use sequential for simple tasks |
