## Prerequisites

| Requirement | Notes |
|-------------|-------|
| Moltbot installed | `moltbot health` |
| Notion account | Free or paid |
| Notion integration | Created at notion.so/my-integrations |

## Step 1: Create Notion Integration

1. Go to [notion.so/my-integrations](https://www.notion.so/my-integrations)
2. Click "New integration"
3. Name: "Moltbot"
4. Select your workspace
5. Capabilities: Check "Read content", "Update content", "Insert content"
6. Click "Submit"
7. Copy the "Internal Integration Token" (starts with `secret_`)

## Step 2: Install Notion Skill

```bash
moltbot skill add notion
```

## Step 3: Add Token to Moltbot

```bash
export NOTION_TOKEN="secret_..."
```

Add to shell profile:
```bash
echo 'export NOTION_TOKEN="secret_..."' >> ~/.zshrc
source ~/.zshrc
```

<HostingCTA context="inline" />

## Step 4: Share Pages with Integration

**Important:** Your integration can only access pages you explicitly share.

1. Open a Notion page or database
2. Click "..." menu ‚Üí "Connections"
3. Find "Moltbot" and click "Connect"

Repeat for each page/database you want Moltbot to access.

## Step 5: Configure

```bash
moltbot configure --section skills.notion
```

Or edit config:
```json
{
  "skills": {
    "notion": {
      "token": "${NOTION_TOKEN}",
      "defaultDatabase": "Tasks",
      "workspacePages": ["Meeting Notes", "Projects", "Tasks"]
    }
  }
}
```

## Database Templates

Set up templates for common actions:

```json
{
  "skills": {
    "notion": {
      "templates": {
        "meeting": {
          "database": "Meeting Notes",
          "properties": {
            "Date": "{{today}}",
            "Type": "Meeting"
          }
        },
        "task": {
          "database": "Tasks",
          "properties": {
            "Status": "To Do",
            "Created": "{{today}}"
          }
        }
      }
    }
  }
}
```

Usage:
```
You: Create a meeting note
Moltbot: [Uses meeting template automatically]
```

## Security Considerations

| Setting | Recommendation |
|---------|----------------|
| Integration scope | Only share needed pages |
| Token storage | Environment variable |
| Sensitive pages | Don't share with integration |

### Limit Access

Only share specific pages:
- ‚úÖ Work databases
- ‚úÖ Meeting notes
- ‚ùå Personal journal
- ‚ùå Salary information

## Troubleshooting

| Issue | Solution |
|-------|----------|
| "Page not found" | Share page with Moltbot integration |
| "Unauthorized" | Check token is correct and not expired |
| "Database not found" | Share database, not just pages in it |
| Properties missing | Check database schema matches config |

**Test connection:**
```bash
moltbot skill test notion search --query="test"
```

**List accessible pages:**
```bash
moltbot skill test notion list-pages
```
