# Moltbot Personal Assistant: Complete Setup Guide

![Moltbot Personal Assistant](/images/articles/clawdbot-personal-assistant.jpg)

## What You'll Build

A personal AI assistant that:
- Sends morning briefings with weather, calendar, and news
- Manages your tasks and reminders
- Answers questions and does research
- Controls smart home devices
- Learns your preferences over time

| Feature | ChatGPT App | Moltbot |
|---------|-------------|---------|

| Requirement | Notes |

## Step 1: Core Setup

If you haven't installed Moltbot yet:

```bash
curl -fsSL https://molt.bot/install.sh | bash
moltbot onboard
```

Add your Claude API key:
```bash
export ANTHROPIC_API_KEY="sk-ant-..."
```

## Step 2: Connect Telegram

This lets you message your AI from anywhere.

1. Open Telegram, search for @BotFather
2. Send `/newbot`, follow prompts
3. Copy the token

```bash
moltbot configure --section channels.telegram
```

Test it:
```bash
moltbot gateway
```

Send a message to your bot on Telegram!

## Privacy & Security

| Setting | Recommendation |
|---------|----------------|
| DM policy | Set to "pairing" mode |
| Data storage | Local only (no cloud sync) |
| API provider | Claude for best privacy policy |
| Sensitive data | Don't share passwords, SSN |

All data stays on your machine. Conversations go to Claude API but aren't stored for training.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Morning briefing not arriving | Check automation time and timezone |
| Skills not working | Run `moltbot skill list` to verify installed |
| Slow responses | Check API key and network connection |
| Memory not persisting | Verify `~/.moltbot/memory/` exists |

**Check status:**
```bash
moltbot doctor
moltbot gateway status
```
