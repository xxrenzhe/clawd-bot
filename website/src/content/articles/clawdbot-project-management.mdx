# Moltbot Project Management: Tasks, Sprints, Reports

![Moltbot Project Management](/images/articles/clawdbot-project-management.jpg)

## Project Management Features

| Feature | Description | Example |
|---------|-------------|---------|
| Task creation | Create tasks via chat | "Create task: Fix login bug" |
| Sprint planning | Plan and manage sprints | "What's in this sprint?" |
| Status reports | Generate project updates | "Weekly status report" |
| Blocker tracking | Identify and escalate | "Show all blocked tasks" |
| Team coordination | Assign and track work | "What's Sarah working on?" |

<HostingCTA context="setup" />

## Prerequisites

| Requirement | Notes |
|-------------|-------|
| Moltbot installed | `moltbot health` |
| Project tool integration | Jira, Linear, or Notion |
| Team configured | User mappings set |

## Quick Setup

```bash
moltbot skill add jira        # or
moltbot skill add linear      # or
moltbot skill add notion
```

## Automation

### Daily Standup Reminder

```json
{
  "automations": {
    "standup_reminder": {
      "schedule": "0 9 * * 1-5",
      "channel": "slack",
      "slackChannel": "#team-standup",
      "prompt": "Generate standup summary and prompt for updates"
    }
  }
}
```

### Weekly Report Automation

```json
{
  "automations": {
    "weekly_report": {
      "schedule": "0 17 * * 5",
      "channel": "email",
      "recipients": ["stakeholders@company.com"],
      "prompt": "Generate weekly status report for Sprint {{current_sprint}}"
    }
  }
}
```

## Configuration

```json
{
  "skills": {
    "project": {
      "provider": "jira",
      "project": "PROJ",
      "defaultAssignee": "auto",
      "sprintBoard": "Team Board",
      "notifications": {
        "blockers": true,
        "overdue": true,
        "statusChanges": false
      }
    }
  }
}
```

| Issue | Solution |
|-------|----------|

| Wrong sprint | Verify board configuration |
| Missing team members | Update user mappings |
| Slow queries | Use project filters |
