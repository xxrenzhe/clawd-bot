# Moltbot on Raspberry Pi: Low-Power AI Setup

![Moltbot running on Raspberry Pi](/images/articles/clawdbot-raspberry-pi.jpg)

## Why Raspberry Pi?

Raspberry Pi is the most affordable way to run Moltbot (formerly Clawdbot) 24/7:

| Metric | Pi 4 (4GB) | Pi 5 (8GB) | VPS (Basic) |
|--------|------------|------------|-------------|
| Hardware cost | $55 | $80 | — |
| Monthly electricity | ~$1 | ~$1.50 | — |
| Monthly hosting | — | — | $5-10 |
| Performance | Good | Excellent | Good |
| 1-year cost | ~$67 | ~$98 | $60-120 |

The Pi pays for itself within a year compared to VPS hosting.

## Hardware Requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| Raspberry Pi | Pi 4 (2GB) | Pi 5 (4GB+) |
| microSD card | 16GB Class 10 | 32GB+ A2 rated |
| Power supply | Official 15W | Official 27W (Pi 5) |
| Cooling | Passive heatsink | Active cooler |
| Case | Any | With ventilation |

**Recommended kit:** Raspberry Pi 5 8GB + official cooler + 64GB A2 microSD

## Step 1: Flash Raspberry Pi OS

1. Download [Raspberry Pi Imager](https://www.raspberrypi.com/software/)
2. Choose OS: **Raspberry Pi OS (64-bit)** — the 64-bit version is required
3. Click the gear icon ⚙️ to pre-configure:
   - Set hostname (e.g., `moltbot`)
   - Enable SSH
   - Set username/password
   - Configure Wi-Fi
4. Flash to microSD card

## Step 2: Initial Setup

Insert the microSD card and power on. Wait 2-3 minutes for first boot.

**Connect via SSH:**
```bash
ssh pi@moltbot.local
# or use IP address: ssh pi@192.168.1.xxx
```

**Update the system:**
```bash
sudo apt update && sudo apt upgrade -y
sudo reboot
```

## Step 3: Install Node.js 22 (ARM64)

```bash
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt-get install -y nodejs
```

Verify:
```bash
node --version  # Should show v22.x
node -p "process.arch"  # Should show arm64
```

## Step 4: Install Moltbot

```bash
curl -fsSL https://molt.bot/install.sh | bash
```

Or via npm:
```bash
npm install -g moltbot
```

## Step 5: Run Setup

```bash
moltbot onboard
```

Get your API key from [console.anthropic.com](https://console.anthropic.com).

Test:
```bash
moltbot health
```

## Step 6: Create systemd Service

This ensures Moltbot starts on boot and restarts if it crashes.

**Create service file:**
```bash
sudo nano /etc/systemd/system/moltbot.service
```

**Paste:**
```ini
[Unit]
Description=Moltbot AI Gateway
After=network.target

[Service]
Type=simple
User=pi
Environment=ANTHROPIC_API_KEY=sk-ant-your-key-here
ExecStart=/usr/bin/moltbot gateway
Restart=always
RestartSec=10
StandardOutput=journal
StandardError=journal

[Install]
WantedBy=multi-user.target
```

Replace `pi` with your username and add your API key.

**Enable and start:**
```bash
sudo systemctl daemon-reload
sudo systemctl enable moltbot
sudo systemctl start moltbot
```

**Check status:**
```bash
sudo systemctl status moltbot
```

## Step 7: Performance Optimization

### Reduce memory usage

Edit `/boot/config.txt`:
```bash
sudo nano /boot/config.txt
```

Add:
```ini
# Reduce GPU memory (headless server)
gpu_mem=16
```

Reboot:
```bash
sudo reboot
```

### Use swap file (for 2GB Pi)

```bash
sudo dphys-swapfile swapoff
sudo nano /etc/dphys-swapfile
# Set CONF_SWAPSIZE=2048
sudo dphys-swapfile setup
sudo dphys-swapfile swapon
```

### Overclock Pi 5 (optional)

Edit `/boot/config.txt`:
```ini
arm_freq=2800
over_voltage=4
```

**Note:** Requires good cooling. Monitor temperature with `vcgencmd measure_temp`.

## Connect Telegram

```bash
moltbot configure --section channels.telegram
```

1. Open Telegram, search for @BotFather
2. Send `/newbot`, follow prompts
3. Copy the token when prompted

## Remote Access

### Option A: Tailscale (easiest)

```bash
curl -fsSL https://tailscale.com/install.sh | sh
sudo tailscale up
```

Access your Pi from anywhere via its Tailscale IP.

### Option B: SSH with key authentication

```bash
# On your local machine
ssh-copy-id pi@moltbot.local
```

## Monitoring

**View logs:**
```bash
sudo journalctl -u moltbot -f
```

**Check CPU temperature:**
```bash
vcgencmd measure_temp
```

**Monitor resources:**
```bash
htop
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Service fails to start | Check `journalctl -u moltbot -e` for errors |
| High CPU temperature | Add active cooling, reduce overclock |
| Out of memory | Add swap, upgrade to 4GB+ model |
| SD card corruption | Use quality A2 card, consider SSD via USB |
| Network drops | Use Ethernet instead of Wi-Fi |

**Restart service:**
```bash
sudo systemctl restart moltbot
```

**Update Moltbot:**
```bash
npm update -g moltbot
sudo systemctl restart moltbot
```

## Comparison: Pi 4 vs Pi 5

| Feature | Pi 4 (4GB) | Pi 5 (8GB) |
|---------|------------|------------|
| CPU | 4-core 1.8GHz | 4-core 2.4GHz |
| RAM | 4GB | 8GB |
| Price | ~$55 | ~$80 |
| Response time | Good | Faster |
| Browser automation | Limited | Possible |
| Power draw | 3-5W | 4-8W |

**Recommendation:** Pi 5 (4GB or 8GB) for the best experience. Pi 4 works for basic use.

## Power Consumption

| State | Pi 4 | Pi 5 |
|-------|------|------|
| Idle | 2.7W | 3.5W |
| Moltbot running | 3-4W | 4-6W |
| Monthly cost | ~$1 | ~$1.50 |

*At $0.15/kWh average electricity cost*
