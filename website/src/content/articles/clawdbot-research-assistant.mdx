## Configuration

```json
{
  "skills": {
    "research": {
      "defaultDepth": "comprehensive",
      "includeSources": true,
      "factCheck": true,
      "maxSearchResults": 10
    }
  }
}
```

### Research Modes

| Mode | Description | Use Case |
|------|-------------|----------|
| `quick` | 2-3 sources, summary | Quick answers |
| `standard` | 5-7 sources, analysis | General research |
| `comprehensive` | 10+ sources, deep dive | Important decisions |

## Best Practices

1. **Be specific** - "AI regulation" ‚Üí "EU AI Act 2026 compliance requirements for chatbots"

2. **Specify format** - "Give me a comparison table" or "List pros and cons"

3. **Request sources** - "Include sources" ensures verifiable information

4. **Iterate** - "Tell me more about point 3" to dive deeper

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Outdated information | Add "latest" or "2026" to query |
| No sources cited | Enable `includeSources` in config |
| Too shallow | Use "comprehensive" depth |
| PDF not reading | Check file size (max 10MB default) |
