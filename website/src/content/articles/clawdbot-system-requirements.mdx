# Moltbot System Requirements: Hardware, OS, Dependencies

![Moltbot System Requirements](/images/articles/clawdbot-system-requirements.jpg)

## Quick Requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| CPU | 1 core | 2+ cores |
| RAM | 512 MB | 2 GB+ |
| Storage | 200 MB | 1 GB+ |
| Node.js | 22+ | 22 LTS |
| OS | macOS 12+, Ubuntu 20+, Windows 10+ (WSL2) | Latest |

## Check Your System

```bash
# Check Node.js version
node --version
# Required: v22.0.0 or higher

# Check available RAM
free -h  # Linux
sysctl hw.memsize  # macOS

# Check disk space
df -h
```

## Operating System Support

| OS | Supported | Notes |
|----|-----------|-------|
| macOS 12+ | Full | Intel and Apple Silicon |
| Ubuntu 20.04+ | Full | LTS recommended |
| Debian 11+ | Full | Stable branch |
| Fedora 38+ | Full | - |
| Windows 10+ | WSL2 only | Native Windows not supported |
| Raspberry Pi OS | Full | 64-bit required |

### macOS

```bash
# Check macOS version
sw_vers

# Check architecture
uname -m
# arm64 = Apple Silicon, x86_64 = Intel
```

**Apple Silicon (M1/M2/M3/M4):** Fully supported, excellent performance.

**Intel Macs:** Fully supported, ensure adequate RAM.

### Linux

```bash
# Check distribution
cat /etc/os-release

# Check architecture
uname -m
# x86_64 = 64-bit Intel/AMD, aarch64 = ARM64
```

### Windows (WSL2 Required)

Native Windows not supported. Install WSL2:

```powershell
# In PowerShell as Administrator
wsl --install -d Ubuntu

# Then in WSL2
curl -fsSL https://molt.bot/install.sh | bash
```

|-----------|-------------|

|-----------|-------------|

|-----------|-------------|

## Hardware Recommendations

### Mac Mini M4 (Best for 24/7)

| Spec | Recommendation |
|------|----------------|
| Model | Mac Mini M4 (base) |
| RAM | 16 GB (or 8 GB for light use) |
| Storage | 256 GB |
| Cost | ~$599 |
| Power | ~5W idle |

Pros: Native macOS, iMessage support, low power, excellent performance.

### Raspberry Pi 5

| Spec | Recommendation |
|------|----------------|
| Model | Raspberry Pi 5 |
| RAM | 4 GB or 8 GB |
| Storage | 32 GB+ SD card or SSD |
| Cost | ~$60-80 |
| Power | ~3W idle |

Pros: Very low cost, low power, always on.

Cons: Limited performance, no browser automation.

## Node.js Installation

### Using nvm (Recommended)

```bash
# Install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Restart terminal, then:
nvm install 22
nvm use 22
nvm alias default 22

# Verify
node --version
```

### Direct Installation

**macOS:**
```bash
brew install node@22
```

**Ubuntu/Debian:**
```bash
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt-get install -y nodejs
```

**Fedora:**
```bash
sudo dnf install nodejs
```

## Optional Dependencies

| Dependency | Required For |
|------------|--------------|

| Redis | High-performance caching |

### Install Optional Dependencies

```bash
# Docker
curl -fsSL https://get.docker.com | sh

# Ollama (local AI)
curl -fsSL https://ollama.com/install.sh | sh

# Chromium (browser automation)
moltbot skill add browser
```

## Network Requirements

| Port | Usage | Direction |
|------|-------|-----------|
| 18789 | Gateway (local only) | Inbound |
| 443 | API calls to Claude/OpenAI | Outbound |
| 11434 | Ollama (if local AI) | Localhost |

### Firewall Rules

```bash
# Allow outbound HTTPS (usually default)
sudo ufw allow out 443

# Block gateway from external (recommended)
sudo ufw deny 18789
```

## Storage Requirements

| Data Type | Size | Growth |
|-----------|------|--------|
| Installation | 150 MB | Fixed |
| Configuration | < 1 KB | Minimal |
| Conversation memory | 1-100 MB | Per usage |
| Cache | 10-500 MB | Configurable |
| Logs | 10-50 MB | With rotation |

### Manage Storage

```bash
# Check moltbot storage usage
du -sh ~/.moltbot/*

# Clear cache
moltbot cache clear

# Prune old conversations
moltbot memory prune --older-than 30d
```

## Troubleshooting

| Issue | Check | Fix |
|-------|-------|-----|
| "Node.js too old" | `node --version` | Install Node.js 22+ |
| Out of memory | `free -h` | Add swap or upgrade RAM |
| Disk full | `df -h` | Clear cache, prune memory |
| Permission errors | File ownership | `chown -R $USER ~/.moltbot` |
| Slow performance | CPU/RAM usage | Upgrade hardware or reduce features |
