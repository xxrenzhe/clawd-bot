## Event Triggers

### Email Triggers

```json
{
  "triggers": {
    "urgent_email": {
      "event": "email.received",
      "conditions": {
        "from": ["boss@company.com", "ceo@company.com"],
        "subject_contains": ["urgent", "asap"]
      },
      "action": {
        "notify": "telegram",
        "message": "üö® Urgent email from {{sender}}: {{subject}}"
      }
    }
  }
}
```

```json

      "schedule": "0 3 * * *",
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Automation not running | Check gateway is running 24/7 |
| Wrong timezone | Set `timezone` in automation config |
| Trigger not firing | Verify conditions match exactly |
| Workflow failing | Check logs: `moltbot logs --filter automations` |

**Debug mode:**
```bash
moltbot automations list --verbose
moltbot automations test morning_briefing
```
