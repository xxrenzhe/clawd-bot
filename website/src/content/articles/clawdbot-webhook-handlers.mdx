## Incoming Webhooks

### Register Handler

```json
{
  "webhooks": {
    "incoming": {
      "github": {
        "path": "/webhooks/github",
        "secret": "${GITHUB_WEBHOOK_SECRET}",
        "events": ["push", "pull_request", "issues"]
      },
      "stripe": {
        "path": "/webhooks/stripe",
        "secret": "${STRIPE_WEBHOOK_SECRET}",
        "events": ["payment_intent.succeeded", "customer.created"]
      }
    }
  }
}
```

### Handler Code

```javascript
// ~/.moltbot/webhooks/github.js
export default {
  name: 'github',

  async onPush(payload, context) {
    const { repository, commits, pusher } = payload;

    await context.notify({
      channel: 'telegram',
      chatId: context.config.notifyChat,
      message: `üì¶ ${pusher.name} pushed ${commits.length} commits to ${repository.name}`
    });
  },

  async onPullRequest(payload, context) {
    const { action, pull_request, repository } = payload;

    if (action === 'opened') {
      // Trigger AI review
      await context.agent.chat({
        prompt: `Review this PR: ${pull_request.title}\n\n${pull_request.body}`,
        context: { prUrl: pull_request.html_url }
      });
    }
  },

  async onIssues(payload, context) {
    const { action, issue } = payload;

    if (action === 'opened') {
      await context.notify({
        channel: 'discord',
        channelId: context.config.issueChannel,
        message: `üêõ New issue: ${issue.title}`
      });
    }
  }
};
```

## Event Types

### Message Events

| Event | Trigger |
|-------|---------|
| `message.received` | New message from user |
| `message.sent` | Response sent to user |
| `message.error` | Message processing failed |

### Agent Events

| Event | Trigger |
|-------|---------|
| `agent.thinking` | AI processing started |
| `agent.response` | AI response generated |
| `tool.called` | Tool execution started |
| `tool.completed` | Tool execution finished |

### System Events

| Event | Trigger |
|-------|---------|
| `gateway.start` | Gateway started |
| `gateway.stop` | Gateway stopped |
| `error` | Any error occurred |
| `rate_limit` | Rate limit triggered |

## Webhook Security

### Signature Verification

```javascript
// Verify GitHub webhook signature
import crypto from 'crypto';

function verifyGitHubSignature(payload, signature, secret) {
  const expectedSignature = 'sha256=' + crypto
    .createHmac('sha256', secret)
    .update(payload)
    .digest('hex');

  return crypto.timingSafeEqual(
    Buffer.from(signature),
    Buffer.from(expectedSignature)
  );
}
```

## Automation Triggers

### Time-Based

```json
{
  "webhooks": {
    "triggers": {
      "daily-summary": {
        "cron": "0 9 * * *",
        "action": {
          "type": "agent",
          "prompt": "Generate a summary of yesterday's activity"
        }
      },
      "weekly-report": {
        "cron": "0 10 * * MON",
        "action": {
          "type": "webhook",
          "url": "https://api.example.com/generate-report"
        }
      }
    }
  }
}
```

### Event-Based

```json
{
  "webhooks": {
    "triggers": {
      "new-user": {
        "event": "user.first_message",
        "action": {
          "type": "message",
          "template": "Welcome! I'm Moltbot, your AI assistant."
        }
      },
      "error-alert": {
        "event": "error",
        "filter": { "severity": "critical" },
        "action": {
          "type": "webhook",
          "url": "${ALERT_WEBHOOK_URL}"
        }
      }
    }
  }
}
```

## Webhook Pipeline

### Transform Payloads

```json
{
  "webhooks": {
    "incoming": {
      "custom": {
        "path": "/webhooks/custom",
        "transform": {
          "message": "$.data.text",
          "userId": "$.metadata.sender_id",
          "timestamp": "$.created_at"
        }
      }
    }
  }
}
```

### Filter Events

```json
{
  "webhooks": {
    "outgoing": {
      "important-only": {
        "url": "https://api.example.com/events",
        "filter": {
          "severity": { "$in": ["high", "critical"] },
          "type": { "$ne": "debug" }
        }
      }
    }
  }
}
```

## Troubleshooting

| Issue | Cause | Solution |
|-------|-------|----------|
| 401 Unauthorized | Wrong secret | Check signature/secret |
| 404 Not Found | Wrong path | Verify webhook path |
| 413 Payload Too Large | Exceeded limit | Increase maxPayloadSize |
| Timeout | Slow handler | Add async processing |
| Missing events | Wrong filter | Check event configuration |
