# Moltbot Workflow Optimization: Boost Productivity

## Workflow Patterns

| Pattern | Use Case | Complexity |
|---------|----------|------------|
| Trigger-Action | Event responses | Low |
| Chain | Multi-step processes | Medium |
| Parallel | Independent tasks | Medium |
| Conditional | Decision flows | High |

## Trigger-Action Workflows

### Basic Pattern

```json
{
  "workflows": {
    "email-alert": {
      "trigger": {
        "type": "event",
        "source": "gmail",
        "filter": { "from": "important@client.com" }
      },
      "action": {
        "type": "notify",
        "channel": "telegram",
        "message": "Important email from {{from}}"
      }
    }
  }
}
```

### Common Triggers

| Trigger | Example |
|---------|---------|
| Schedule | `cron:0 9 * * *` |
| Event | `gmail:new_email` |
| Keyword | `message:contains:urgent` |
| Webhook | `webhook:github:push` |

## Chain Workflows

### Sequential Steps

```json
{
  "workflows": {
    "meeting-prep": {
      "trigger": "cron:0 8 * * 1-5",
      "steps": [
        {
          "id": "get-meetings",
          "tool": "calendar_today"
        },
        {
          "id": "fetch-docs",
          "tool": "drive_search",
          "input": "{{get-meetings.attendees}}"
        },
        {
          "id": "summarize",
          "tool": "agent_chat",
          "input": "Summarize: {{fetch-docs.content}}"
        },
        {
          "id": "notify",
          "tool": "telegram_send",
          "input": "{{summarize.response}}"
        }
      ]
    }
  }
}
```

### With Error Handling

```json
{
  "workflows": {
    "data-sync": {
      "steps": [
        {
          "id": "fetch",
          "tool": "api_fetch",
          "onError": "skip"
        },
        {
          "id": "process",
          "tool": "transform",
          "onError": {
            "action": "notify",
            "message": "Sync failed: {{error}}"
          }
        }
      ]
    }
  }
}
```

## Parallel Workflows

### Concurrent Execution

```json
{
  "workflows": {
    "morning-briefing": {
      "trigger": "cron:0 7 * * *",
      "parallel": [
        { "tool": "weather_get", "as": "weather" },
        { "tool": "calendar_today", "as": "calendar" },
        { "tool": "email_unread", "as": "emails" },
        { "tool": "news_fetch", "as": "news" }
      ],
      "then": {
        "tool": "agent_chat",
        "prompt": "Create briefing from: {{weather}}, {{calendar}}, {{emails}}, {{news}}"
      }
    }
  }
}
```

## Conditional Workflows

### Decision Flow

```json
{
  "workflows": {
    "ticket-triage": {
      "trigger": "zendesk:new_ticket",
      "steps": [
        {
          "id": "classify",
          "tool": "agent_classify",
          "input": "{{ticket.body}}"
        },
        {
          "id": "route",
          "type": "switch",
          "on": "{{classify.category}}",
          "cases": {
            "billing": { "action": "assign", "to": "billing-team" },
            "technical": { "action": "assign", "to": "support-team" },
            "sales": { "action": "assign", "to": "sales-team" },
            "default": { "action": "queue", "to": "general" }
          }
        }
      ]
    }
  }
}
```

## Optimization Tips

### Reduce Latency

| Technique | Benefit |
|-----------|---------|
| Cache frequent queries | 50-90% faster |
| Use parallel execution | 2-3x speedup |
| Pre-fetch predictable data | Instant responses |

### Cache Configuration

```json
{
  "workflows": {
    "market-check": {
      "cache": {
        "enabled": true,
        "ttl": 300,
        "key": "market-data"
      }
    }
  }
}
```

### Rate Limiting

```json
{
  "workflows": {
    "api-sync": {
      "rateLimit": {
        "requests": 100,
        "window": "1m"
      }
    }
  }
}
```

## Troubleshooting

| Issue | Cause | Solution |
|-------|-------|----------|
| Workflow not triggering | Wrong cron syntax | Validate schedule |
| Step timeout | Slow API | Increase timeout |
| Data not passed | Wrong reference | Check variable names |
| Partial execution | Error in chain | Add error handling |
