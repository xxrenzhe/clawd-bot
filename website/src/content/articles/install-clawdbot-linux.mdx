# Install Moltbot on Linux: Ubuntu/Debian Guide

![Installing Moltbot on Linux](/images/articles/install-clawdbot-linux.jpg)

## Quick Install (1 minute)

```bash
# Install Node.js 22
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt-get install -y nodejs

# Install Moltbot
curl -fsSL https://molt.bot/install.sh | bash

# Run setup
moltbot onboard
```

## Prerequisites

| Requirement | Minimum | Tested On |
|-------------|---------|-----------|
| OS | Ubuntu 20.04+ | Ubuntu 22.04, 24.04, Debian 12 |
| Node.js | 22+ | Node.js 22 LTS |
| RAM | 2GB | 4GB recommended |
| Storage | 500MB | 1GB recommended |

## Step 1: Install Node.js 22

**Ubuntu/Debian:**
```bash
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt-get install -y nodejs
```

**Fedora/RHEL:**
```bash
sudo dnf install -y nodejs
```

**Arch Linux:**
```bash
sudo pacman -S nodejs npm
```

**Verify:**
```bash
node --version  # Should show v22.x
```

## Step 2: Install Moltbot

```bash
curl -fsSL https://molt.bot/install.sh | bash
```

Or via npm:
```bash
npm install -g moltbot
```

## Step 3: Run Setup Wizard

```bash
moltbot onboard
```

This creates `~/.moltbot/moltbot.json` and prompts for:
- Anthropic API key (for Claude)
- Optional: Telegram/Discord setup

## Step 4: Add API Key

Get your key from [console.anthropic.com](https://console.anthropic.com).

```bash
# Add to your shell profile
echo 'export ANTHROPIC_API_KEY="sk-ant-..."' >> ~/.bashrc
source ~/.bashrc
```

## Step 5: Verify Installation

```bash
moltbot health
```

Expected output:
```
✓ Gateway healthy
✓ Claude API connected
✓ Config valid
```

## Run as systemd Service (24/7 Operation)

Create a systemd service file:

```bash
sudo nano /etc/systemd/system/moltbot.service
```

Paste:
```ini
[Unit]
Description=Moltbot AI Gateway
After=network.target

[Service]
Type=simple
User=YOUR_USERNAME
Environment=ANTHROPIC_API_KEY=sk-ant-...
ExecStart=/usr/bin/moltbot gateway
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

Replace `YOUR_USERNAME` with your username.

Enable and start:
```bash
sudo systemctl daemon-reload
sudo systemctl enable moltbot
sudo systemctl start moltbot
```

Check status:
```bash
sudo systemctl status moltbot
```

## Connect Telegram

1. Open Telegram, search for @BotFather
2. Send `/newbot`, follow prompts
3. Copy the token

```bash
moltbot configure --section channels.telegram
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| `command not found: moltbot` | Add `~/.npm-global/bin` to PATH |
| `Permission denied` | Fix npm permissions (don't use sudo) |
| `Port 18789 in use` | `sudo lsof -i :18789` and kill process |
| `systemd fails` | Check logs: `journalctl -u moltbot -f` |

**Fix npm permissions:**
```bash
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
```

## Security Notes

- Bind to loopback only: set `gateway.bind: "loopback"` in config
- Use Cloudflare Tunnel or Tailscale for remote access
- Never expose port 18789 to the internet
